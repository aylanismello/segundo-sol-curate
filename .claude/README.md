# Claude Code Project

This directory contains configuration for Claude Code.

## Structure

- `settings.json` - Project settings and MCP server configuration
- `commands/` - Custom slash commands for this project
- `README.md` - This file

## Custom Commands

You can add custom slash commands by creating markdown files in the `commands/` directory.
Each file should contain the prompt you want to execute when the command is called.

Example: `.claude/commands/review.md`
